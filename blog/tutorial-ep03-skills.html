给龙虾

为什么需要Skill？

EP01你搭好了窝，EP02你装好了大脑。但光有大模型还不够——

Skill = 龙虾的专业技能。就像人学了编程才能写代码、学了设计才能做图。Skill让龙虾从"能回答"变成"能执行"。

Step 1 打开Skill商店

1

访问 Ima Claw Skill 商店

浏览器打开：imaclaw.github.io/ima-claw-v2/skills/

或者直接在飞书问你的龙虾：打开Skill商店

你会看到几十个Skill，分三大类：

• 🧠 智能类 — 记忆、任务、调研
• 🎨 创作类 — 图片、视频、音乐
• 🔧 工具类 — 浏览器、飞书、Twitter

Step 2 安装第一个Skill（必装）

🧠

IMA 基础智能 Skill 包

必装 让龙虾真正"懂事"
这是IMA官方出品的基础能力包，相当于给龙虾装上"长期记忆"和"项目管理"能力。没装这个，龙虾每次对话都从零开始，记不住你们之前聊过什么。
记忆系统 — 记住你们的对话历史
项目记忆 — 每个项目独立存档
任务追踪 — 自动记录TODO清单
深度调研 — 联网搜索+报告生成
上下文管理 — 长对话不丢上下文
2a

一键安装

在飞书对龙虾说：

飞书 · Ima Claw
你：安装 ima-intelligence

🦞：正在安装 IMA 基础智能 Skill 包...
✓ 记忆系统
✓ 项目记忆
✓ 任务追踪
🎉 安装完成！现在我可以记住我们的对话了。

或者终端命令：

~ $ openclaw skills install ima-intelligence
2b

验证安装

在飞书问：

飞书 · Ima Claw
你：记住我叫小明，我喜欢吃辣

🦞：记住了，小明喜欢吃辣 👍

你：（过几分钟）我叫什么？

🦞：你叫小明，喜欢吃辣。

如果龙虾能记住，说明记忆系统正常工作了！

Step 3 安装创作Skill（选装）

🦞

IMA 全能创作 All-in-One

强烈推荐 一个Skill = 40+ AI模型
如果你需要龙虾帮你做图、做视频、做音乐，装这一个就够了。集成图片生成（SeeDream、Midjourney、GPT Image）、视频生成（Wan、Kling、Sora、Veo）、音乐生成（Suno）等40+模型。
SeeDream 4.5 — 字节图片旗舰
Midjourney — 艺术风格首选
Wan 2.6 — 开源视频之王
Kling 2.1 — 国产视频最强
Sora — OpenAI视频
Google Veo 3 — 谷歌视频
Suno — AI音乐标杆
积分统一 — 不用到处充值
3

安装全能创作包

在飞书对龙虾说：

飞书 · Ima Claw
你：安装 ima-all-in-one

🦞：正在安装 IMA 全能创作 All-in-One...
✓ 图片生成模块
✓ 视频生成模块
✓ 音乐生成模块
🎉 安装完成！我可以帮你生成图片、视频和音乐了。

或终端命令：

~ $ openclaw skills install ima-all-in-one

💥 翻车时刻：提示"需要API Key"？

原因：IMA创作Skill需要ImaStudio账号和API Key

解决：

1. 去 imastudio.com 注册账号
2. 在个人中心获取 API Key
3. 在OpenClaw配置：openclaw config set ima.api_key YOUR_KEY

🎁 新用户福利：注册送免费积分，够生成几十张图
