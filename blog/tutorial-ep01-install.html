Step 1 找到终端

终端（Terminal）是你和电脑「对话」的窗口。所有安装命令都在这里输入。

A

🍎 Mac 用户

🔍
3种方法打开终端（任选其一）

方法1（最快）：按 ⌘ + 空格，输入 Terminal 或 终端，回车
方法2：打开 Launchpad → 其他 → 终端
方法3：Finder → 应用程序 → 实用工具 → 终端

B

🪟 Windows 用户

⚠️ 官方推荐用 WSL2（Windows子系统）运行 OpenClaw，稳定性更好。

🔍
打开 PowerShell

方法1：按 Win 键，输入 PowerShell，点击打开
方法2：右键桌面左下角Windows图标 → Windows PowerShell

推荐：安装 WSL2（一劳永逸）

PowerShell
PS C:\> wsl --install
Installing: Ubuntu...
重启电脑后，在开始菜单打开 "Ubuntu"

装完 WSL2 后，以后就打开「Ubuntu」应用，所有命令和Mac一样。

Step 2 一键安装

官方推荐用「一键安装脚本」，它会自动检测你的系统、安装需要的依赖、然后启动引导向导。

1

🍎 Mac / Linux / WSL2

在终端里复制粘贴这行，按回车：

Terminal — zsh
~ $ curl -fsSL https://openclaw.ai/install.sh | bash
==> Downloading OpenClaw...
==> Installing Node 22...
🎉 OpenClaw installed successfully!
Running onboarding wizard...

等 2-3 分钟，看到 🎉 就是装好了！脚本会自动启动引导向导。

2

🪟 Windows（原生PowerShell）

如果你不想用WSL，直接在PowerShell里运行：

PowerShell
PS C:\Users\你> iwr -useb https://openclaw.ai/install.ps1 | iex
Downloading OpenClaw for Windows...
✅ Installation complete!

等 2-3 分钟，看到 ✅ 就是装好了。

💥 翻车时刻 #1：命令找不到？

症状：提示 curl: command not found 或 iwr: 无法识别

原因：系统版本太旧，缺少基础工具

解决：换 npm 方式安装（需要先装 Node.js）

npm 安装方式：

Terminal
~ $ npm install -g openclaw@latest
~ $ openclaw onboard --install-daemon

先去 nodejs.org 下载 LTS 版本安装，然后运行上面命令。

Step 3 完成引导向导

安装脚本会自动启动一个叫 openclaw onboard 的引导向导，它会问你几个问题，帮你完成初始配置。

3

引导向导会问什么？

OpenClaw Onboarding
? Choose your AI provider:
> OpenAI (GPT-4)
Anthropic (Claude)
Google Gemini
Skip for now

? Enter your API key: [hidden]

✅ Configuration saved!
Starting gateway...
🦞 Gateway is running on port 3000

建议：如果你还没有 API Key，选 "Skip for now"（稍后再配）。下一课 EP02 会详细讲怎么给龙虾装大脑。

💥 翻车时刻 #2：端口被占用？

症状：提示 Error: Port 3000 is already in use

原因：上次没关干净

解决：

Terminal
~ $ openclaw gateway stop
~ $ openclaw gateway start

Step 4 验证安装

✓

检查状态

打开新终端窗口，输入：

Terminal
~ $ openclaw status
🦞 Gateway: running
Port: 3000
Version: 1.x.x

看到 🦞 Gateway: running，恭喜你！龙虾的窝搭好了 🎉
