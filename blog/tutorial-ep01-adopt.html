Step 1 安装 OpenClaw

OpenClaw 是「龙虾的窝」——你的AI助手住在这里，通过它连接飞书。第一步，我们要把这个窝搭好。

🍎 Mac 用户

1a

打开「终端」应用

🔍
怎么找到终端？（3种方法，任选其一）

方法1：按 ⌘ + 空格，弹出 Spotlight 搜索框，输入 Terminal（或"终端"），回车打开
方法2：打开 Finder → 应用程序 → 实用工具 → 终端
方法3：在 Launchpad（启动台）里搜索"终端"

打开后你会看到一个黑色/白色的窗口，有一个闪烁的光标等你输入。这就是终端——你和电脑"对话"的地方。

1b

先检查有没有 Homebrew

Homebrew 是 Mac 上的"应用商店命令行版"，装软件用的。在终端里输入：

Terminal — zsh
~ $ brew --version

如果看到版本号（如 Homebrew 4.x.x），说明已经有了，跳到下一步。

如果提示 command not found，需要先装 Homebrew：

Terminal — zsh
~ $ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
==> Downloading and installing Homebrew...
==> Installation successful!
🍺 Homebrew is ready.

这一步大概需要 2-3 分钟，等它跑完出现 🍺 就好了。

1c

安装 OpenClaw

还是在终端里，输入这一行然后按回车：

Terminal — zsh
~ $ brew install openclaw
==> Downloading openclaw...
==> Installing openclaw...
🎉 openclaw was successfully installed!

等 1-2 分钟，看到 🎉 就是装好了！

🪟 Windows 用户

2a

先装 Node.js

📥
下载 Node.js

1. 打开浏览器，访问 nodejs.org
2. 点击绿色大按钮 "LTS 推荐版"（左边那个）下载
3. 双击下载的 .msi 文件，一路点"Next"安装
4. 装完后重启电脑（很重要！不重启可能找不到命令）

2b

打开 PowerShell

🔍
怎么找到 PowerShell？

方法1：按 Win 键（键盘左下角 Windows 图标），输入 PowerShell，点击"Windows PowerShell"
方法2：右键桌面左下角的 Windows 图标 → 选择 "Windows Terminal" 或 "PowerShell"

打开后你会看到一个蓝色/黑色的窗口，有闪烁的光标。

2c

安装 OpenClaw

在 PowerShell 里输入：

PowerShell
PS C:\Users\你> npm install -g openclaw
added 142 packages in 45s
✅ openclaw installed successfully

等 1-2 分钟，看到 ✅ 就是装好了。

💥 翻车时刻 #1

症状：输入 brew 或 npm 提示 command not found

原因：Mac 没装 Homebrew / Windows 没装 Node.js，或者装完没重启

解决：回到上面对应步骤装好前置工具。Windows 用户特别注意 装完 Node.js 要重启电脑！

Step 2 启动你的龙虾

OpenClaw 装好了，现在让龙虾"活"过来。

3

初始化配置

还是在刚才的终端/PowerShell 窗口里，输入：

Terminal
~ $ openclaw init
🦞 Welcome to OpenClaw!
Creating workspace at ~/.openclaw/workspace/...
? Choose your AI provider: > OpenAI (recommended)
Claude (Anthropic)
Other

? Enter your API key: sk-xxxx...

✅ Configuration saved!

它会问你几个问题：

• AI 模型选哪个？推荐选 OpenAI 或 Claude（如果你有API Key的话）
• API Key 在哪拿？去 OpenAI后台 或 Claude后台 创建一个
• 不知道选什么？一路按回车用默认值也行

4

启动网关（龙虾开机！）

输入这行命令让龙虾正式上线：

Terminal
~ $ openclaw gateway start
Starting OpenClaw gateway...
🦞 Gateway started on port 3000
📱 Feishu/Lark: 发送 start a1b2c3d4 到飞书机器人完成配对

⚠️ 重要：这个终端窗口要一直开着！关掉 = 龙虾下线。可以把它最小化，但别关。

💥 翻车时刻 #2

症状：提示 Error: Port 3000 is already in use

原因：上次没关干净，端口还被占着

解决：先停再启：

Terminal
~ $ openclaw gateway stop
~ $ openclaw gateway start

Step 3 和飞书配对

现在让龙虾连上你的飞书，这样你就能在飞书里直接和它聊天了。

5

复制配对码

看刚才终端窗口里的这行：

Terminal
📱 Feishu/Lark: 发送 start a1b2c3d4 到飞书机器人完成配对

把那个 8位字符（上面绿色下划线部分）复制下来。每次启动都会生成新的码，10分钟有效。

6

在飞书里完成配对

💬
怎么找到机器人？

1. 打开飞书App（手机或电脑都行）
2. 点击顶部搜索栏
3. 搜索「Ima Claw」或「OpenClaw」
4. 找到带 🦞 图标的官方机器人，点击进入对话
5. 在对话框里发送：

飞书对话 · Ima Claw
你：start a1b2c3d4

🦞：配对成功！我是你的AI助手，你可以叫我…
（龙虾会开始自我介绍，问你它该叫什么名字）

看到龙虾回复了？🎉 恭喜，配对成功！

💥 翻车时刻 #3

症状：发了 start 命令，龙虾没反应

可能原因：

• 配对码过期了（超过10分钟）→ 回终端重启 openclaw gateway restart 拿新码
• 终端窗口关了 → 龙虾已下线，重新 openclaw gateway start
• 搜到的不是官方机器人 → 确认名字是 "Ima Claw" 且有官方标识

😮‍💨 嫌这步麻烦？Cloud版不用配对，扫码即用 → 预约Cloud版
