为什么写这篇教程？

上个月，我和爪爪（我的AI助手）连续肝了52小时，从零搭建了一个48页的网站。很多人问我：「我也想用AI帮我干活，但完全不会代码，怎么开始？」

这篇教程就是为你们写的。不需要编程基础，跟着做，10分钟后你的飞书就会多一个能听懂话、会干活的员工。

Step 1 安装 OpenClaw

想象一下：OpenClaw 是「龙虾的窝」。你的AI助手（龙虾）住在这里，通过它和飞书对话。

1

Mac 用户

打开终端（Terminal），复制粘贴这行：

brew install openclaw

等它跑完，看到 🎉 就是装好了。

2

Windows 用户

打开 PowerShell，复制粘贴这行：

npm install -g openclaw

需要提前装好 Node.js。装完后重启 PowerShell。

💥 翻车时刻 #1

我装的时候提示 command not found，原来是 Homebrew 没装。

解决：先装 Homebrew：/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Step 2 启动你的龙虾

3

初始化配置

输入：

openclaw init

一路回车（选默认配置就行）。这会创建一个工作文件夹 ~/.openclaw/workspace/，你的龙虾会住在这里。

4

启动网关

输入：

openclaw gateway start

看到 Gateway started 就是成功了。这个窗口要保持开着，关了龙虾就下线了。

💥 翻车时刻 #2

我启动时提示端口被占用，原来是上次没关干净。

解决：先 openclaw gateway stop，再重新 start。

Step 3 和飞书配对

现在让龙虾能「看到」你的飞书消息。

5

获取配对码

在启动网关的窗口里，找到这样一行：

Feishu/Lark: 发送 start xxxxxxxx 到飞书机器人

复制那个 xxxxxxxx（8位字符）。

6

飞书里完成配对

1. 打开飞书，搜索「OpenClaw」官方应用
2. 或者直接在飞书里 @Ima Claw 机器人
3. 发送：start xxxxxxxx（把你复制的8位码填进去）

看到「配对成功」就搞定了！

💥 翻车时刻 #3

配对码过期了！原来码只有10分钟有效期。

解决：重启网关生成新码，或者等Cloud版（不用配对，扫码就行）。
