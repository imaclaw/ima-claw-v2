🍔 汉堡菜单变成了一个"按钮"

🍔 汉堡菜单之谜

Pricing 页面的导航汉堡图标长得像个按钮——有边框有背景。在其他页面是干净的三条杠 ☰。

原因：缺少 .hamburger{background:none;border:none} 的基础样式。浏览器给 button 加了默认样式。

最后用 inline style 强制解决——因为 GitHub Pages 的缓存实在太顽强了。

🏆 今日最佳 Bug

🔍 CSS --blue 变量失踪案

症状：Plus 卡片标签和按钮全部透明
排查时间：0.5 秒（grep 一下就知道了）
修复时间：3 秒
影响范围：整个 pricing 页面所有蓝色元素
根因：新页面的 :root 里忘了定义 --blue 变量
教训：变量不会自己从别的文件飞过来。

🔍 找到了吗？"入门首选"标签和"立即订阅"按钮都透明了
