什么是"大脑"？

EP01你搭好了窝，但里面空空如也。现在要给龙虾装一个"大脑"——大语言模型（LLM）。

简单说：大脑 = 云端的AI服务。你的龙虾通过API调用这些服务，才能理解你说的话、给出聪明的回复。

Step 1 选大脑：模型对比

市面上有很多AI模型，选哪个？下表按国内访问便利性和性价比排序：

模型特点价格国内访问推荐指数
Claude (Anthropic)推理强，代码好，最"懂事"贵需代理⭐⭐⭐⭐⭐
GPT-4 (OpenAI)综合能力最强，生态最全贵需代理⭐⭐⭐⭐⭐
Gemini (Google)免费额度多，支持超长上下文低需代理⭐⭐⭐⭐
Kimi K2.5 (Moonshot)国产最强，200万字长文本中✅直接访问⭐⭐⭐⭐⭐
DeepSeek V3开源黑马，性价比高低✅直接访问⭐⭐⭐⭐
豆包 (火山引擎)字节出品，中文好低✅直接访问⭐⭐⭐⭐
Ollama (本地)完全免费，隐私好，需高配电脑免费本地运行⭐⭐⭐
💡 我的建议

新手首选 Kimi K2.5 —— 国产，不用代理，价格便宜，200万字长文本能力无敌。去 platform.moonshot.cn 注册就能用。

进阶选 Claude —— 如果你有一定技术能力，能搞定代理，Claude 的代码能力和推理能力确实最强。

Step 2 获取API Key

选定模型后，需要去官方平台注册账号，获取 API Key（密钥）。这是你和AI服务之间的"密码"。

A

Kimi (Moonshot) — 推荐

1. 访问 platform.moonshot.cn

2. 用手机号注册，实名认证

3. 进入「API Key管理」→「新建API Key」

4. 复制那串以 sk- 开头的密钥

💰 新用户免费送15元额度，够用一阵子

B

Claude (Anthropic)

1. 访问 console.anthropic.com（需代理）

2. 用Google账号或邮箱注册

3. 进入 Settings → API Keys → Create Key

4. 复制密钥（同样以 sk- 开头）

💰 新用户送5美元额度，但 Claude API 价格较贵

C

其他模型

• OpenAI: platform.openai.com（需代理+国外信用卡）

• Gemini: aistudio.google.com（需代理，免费额度多）

• DeepSeek: platform.deepseek.com（国内直接访问）

• 豆包: console.volcengine.com（字节火山引擎）

Step 3 配置到OpenClaw

拿到API Key后，告诉OpenClaw用哪个模型。

1

方式一：引导向导（推荐）

如果你还没运行过 openclaw onboard，现在运行：

Terminal
~ $ openclaw onboard

? Choose your AI provider:
> OpenAI (GPT-4)
Anthropic (Claude)
Google Gemini
Moonshot (Kimi)
...

? Enter your API key: （粘贴你的sk-xxx）

✅ Model configured!

跟着向导选模型、贴API Key，搞定。

2

方式二：手动命令

如果你已经跑过onboard想换模型：

Terminal
~ $ openclaw models set moonshot/kimi-k2.5
✅ Model set to moonshot/kimi-k2.5

查看所有可用模型：

~ $ openclaw models list

Step 4 测试大脑

🧠

验证配置

在飞书对龙虾说：

飞书 · Ima Claw
你：你是谁？用的什么模型？

🦞：我是你的AI助手，正在使用 Kimi K2.5 模型为你服务。

如果龙虾能正常回复，说明大脑配置成功！🎉

💥 翻车时刻：提示"API Key无效"？

原因1：Key复制错了，或者有多余空格

原因2：额度用完了（Kimi免费15元，Claude免费5刀，用完要充值）

原因3：国内访问国外API被墙（OpenAI/Claude/Gemini需要代理）

解决：换国产模型（Kimi/DeepSeek/豆包），或者检查代理设置。
