使用 MiniMax

⭐ 简单 ⏱ 约 2 分钟 🔑 ImaClaw
💡 Ima Claw 已封装所有模型。你不需要单独去 MiniMax 注册账号或获取 API Key。只需一个 ImaClaw 账号，即可使用包括 MiniMax 在内的所有 30+ 模型。

📋 前置条件

  • 已完成 Ima Claw 安装（还没装？去领养 →）
  • ImaStudio 账号（点击注册，免费 →）
STEP 1

注册 ImaStudio 账号

访问 imastudio.com，点击注册。支持邮箱和 Google 账号快速注册。

注册完全免费，无需信用卡。

STEP 2

获取 ImaClaw API Key

登录后进入 Settings → API Keys，点击创建新 Key。

复制你的 API Key，格式类似：

# 你的 ImaClaw API Key（示例）
imaclaw_XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
⚠️ API Key 只显示一次，请立即复制保存。如果丢失，可以删除旧 Key 重新创建。
STEP 3

填入龙虾配置

编辑配置文件，填入你的 ImaClaw API Key：

nano ~/.openclaw/openclaw.json

在配置文件中设置：

{
  "ai": {
    "provider": "imaclaw",
    "apiKey": "你的 ImaClaw API Key"
  }
}

保存后重启：

openclaw gateway restart
STEP 4

选择模型并验证

配置完成后，你可以在对话中随时切换模型。给龙虾发一条消息测试：

嘿，你好！你是谁？

如果龙虾回复了你，说明一切就绪！🎉

想切换到 MiniMax？直接告诉龙虾：

帮我切换到 MiniMax
