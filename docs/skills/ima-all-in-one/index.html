🔧 安装方式

✅ 官方预装技能包 — 已随 Ima Claw 一起安装，无需额外操作。确保你有 ImaStudio 账号即可使用全部能力。

如果需要手动安装或重新安装：

安装完成后，重启龙虾生效：

❓ 常见问题

这个包和单独的技能有什么区别？▾

技能包是多个相关技能的集合，一键安装即可获得全部能力。你也可以单独安装/关闭包内的任意技能。

需要额外付费吗？▾

技能本身免费，使用时消耗的 AI 模型调用按 ImaClaw API 用量计费。

可以只用其中一部分功能吗？▾

当然可以。龙虾会根据你的需求自动调用合适的子技能，你也可以登录后在管理页面关闭不需要的部分。
