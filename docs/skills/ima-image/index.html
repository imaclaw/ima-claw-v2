📋 关于这个 Skill

10+ 模型聚合的图片生成工具，支持文生图、图生图，一句话出图。

支持模型：SeeDream 4.5 · GPT Image · Midjourney · Imagen 4 · Nano Banana Pro

功能

✨ 核心能力

  • 10+ 图片模型随时切换
  • 文生图（Text to Image）
  • 图生图（Image to Image）
  • 自动选择最佳模型
  • 对比出图（多模型同时生成）
