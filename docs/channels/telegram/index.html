接入 Telegram

⭐ 简单 ⏱ 约 2 分钟 💰 免费

📋 前置条件

  • 已安装 Telegram（手机或电脑端均可）
  • 已完成 Ima Claw 安装（还没装？去领养 →）
  • 无需付费，无需信用卡
STEP 1

找到 @BotFather

打开 Telegram，在搜索栏输入 @BotFather，点击进入对话。

BotFather 是 Telegram 官方的机器人管理工具，所有 Bot 都通过它创建。

💡 认准官方标识：名字旁边有蓝色 ✓ 认证徽标的才是真的 BotFather。
STEP 2

创建你的 Bot

在 BotFather 对话中，发送以下命令：

/newbot

BotFather 会问你两个问题：

1. Bot 的显示名称 — 你希望在聊天中显示的名字，比如 我的龙虾 或 My Claw

2. Bot 的用户名 — 必须以 bot 结尾，比如 my_lobster_bot

STEP 3

拿到 Bot Token

创建成功后，BotFather 会返回一段 Token，格式类似：

# 你的 Bot Token（示例）
7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
⚠️ 这个 Token 是你 Bot 的密钥，不要分享给任何人！如果泄露了，可以在 BotFather 中用 /revoke 命令重新生成。
STEP 4

填入龙虾配置

回到你的终端，在 Ima Claw 安装目录下编辑配置文件：

# 方式一：如果安装向导还在运行，直接粘贴 Token

# 方式二：手动编辑配置文件
nano ~/.openclaw/openclaw.json

找到 channels 部分，添加 Telegram 配置：

{
  "channels": {
    "telegram": {
      "enabled": true,
      "token": "你的Bot Token粘贴在这里"
    }
  }
}

保存后，重启 Ima Claw：

openclaw gateway restart
STEP 5

验证连接 ✅

打开 Telegram，搜索你刚创建的 Bot 用户名，发一条消息：

嘿，你好！你是谁？

如果龙虾回复了你，说明一切就绪！🎉

🎉

恭喜！你的 Ima 龙虾已上线 Telegram

现在可以随时随地用 Telegram 和你的 Ima 龙虾对话了

💡 进阶设置：你还可以配置群聊模式（让龙虾加入群组）、设置管理员白名单、自定义触发词等。详见 完整文档 →

❓ 常见问题

Bot 创建成功了但龙虾没回复？ ▾

检查以下几点：

1. Token 是否正确粘贴（没有多余空格）

2. Ima Claw 是否已重启（openclaw gateway restart）

3. 查看日志：openclaw gateway logs 找报错信息

可以让多个人用同一个 Bot 吗？ ▾

可以！默认情况下任何人都可以和你的 Bot 对话。如果想限制只有特定用户可用，在配置中添加 allowFrom 白名单。

可以把 Bot 加到群组里吗？ ▾

可以！把 Bot 邀请进群后，默认只在被 @提及 时回复。你也可以在配置中设置 groupPolicy 来调整群聊行为。

Token 泄露了怎么办？ ▾

在 BotFather 中发送 /revoke，选择你的 Bot，会生成新 Token。旧 Token 立即失效。然后更新配置文件并重启。
