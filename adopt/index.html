❓ 常见问题

真的免费吗？
Ima Claw 本身完全免费开源。你需要自己提供 AI 模型的 API Key（大部分供应商有免费额度）。龙虾跑在你自己的电脑上，数据也在你自己手里。
需要什么样的电脑？
只要能跑 Node.js 就行——MacBook、Windows 笔记本、Linux 服务器、甚至树莓派都可以。不需要 GPU，因为 AI 推理跑在云端（通过你的 API Key）。
和 ChatGPT 有什么区别？
ChatGPT 是个聊天窗口。龙虾是你的私人 AI 助手——它能记住你、自动执行任务、连接你的工具（日历、邮件、社媒）、24 小时待命，还能学会你教它的任何新技能。
数据安全吗？
龙虾跑在你自己的机器上，数据不经过任何第三方服务器。它的记忆、对话历史、你的文件全部存在本地。你可以随时查看、删除、导出。
IMA Studio 的技能需要额外付费吗？
IMA Studio 的图片、视频、音乐生成走积分制，注册就送免费积分。搭配龙虾使用，一句话就能生成专业级内容。了解 IMA Studio →
遇到问题怎么办？
加入 IMA 社区 提问，或者查看 Ima Claw 文档。龙虾的开发者和用户社区非常活跃！
